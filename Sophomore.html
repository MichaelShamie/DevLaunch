Encapsulation

I used access modifiers like private and public to protect data and keep internal logic hidden from outside classes. Getters and setters allowed safe interaction with private variables.
Definition: Hiding data so only the right code can access or change it. Example:

  • Use private to hide a variable.
  • Use getName() and setName() to safely access it.
