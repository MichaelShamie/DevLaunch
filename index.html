About Me Freshman Sophomore Junior

Welcome to My Portfolio

Hi, my name is Ibrahim, and this is a summary of the classes I’ve taken during my three years of high school. Each year has helped me learn more about computers, technology, and how things work behind the scenes.
Each of these classes taught me something different and helped me figure out that I want to go into cybersecurity in the future. Now I’ll show you more about what I did each year.

Freshman Year

Java Programming: This was my first real coding class. I learned Java, which is a programming language used in a lot of apps, websites, and games. I learned the basics like:

  1. How to print messages on the screen
  2. How to use variables to store numbers or words
  3. How to make decisions using if and else
  4. How to repeat tasks using loops
Engineering with OnShape: In this class, I used a tool called OnShape, which lets you build 3D models of objects. It was kind of like building things in Minecraft but more technical. I learned how to:
  1. Sketch shapes and turn them into 3D parts
  2. Cut and join pieces together
  3. Make full designs of things like tools and parts
  4. This class helped me understand how real engineers design machines, buildings, and products. It taught me how to think in 3D and pay attention to small details.

Sophomore Year

Object-Oriented Programming (OOP): In my second year, I went deeper into coding. I learned Object-Oriented Programming, which is a way to write code that’s more organized and easier to fix or reuse. I learned how to:

  1. Make classes, which are like blueprints for objects
  2. Create objects that can do things and hold data
  3. Use inheritance, where one class can take things from another
  4. Keep parts of code private or public to protect it

Junior Year

Web Design: In this class, I learned how websites are built from scratch. I used HTML to add content like text, images, and links, and CSS to style everything with colors, fonts, and layouts. Some of the things I learned:

  1. How to make menus and sections with div tags
  2. How to use Flexbox and Grid to line things up on the pages
  3. How to change styles when someone hovers over a button
  4. How to make websites look good on phones and computers
I really enjoyed designing pages and seeing how my ideas came to life on the screen.
Cybersecurity: This was one of my favorite classes. I learned how to protect computers and networks from hackers and viruses. We learned about:
  1. Different types of cyber attacks like phishing and malware
  2. Password safety and how hackers guess passwords
  3. How firewalls and antivirus software protect systems
  4. Ethical hacking (using hacking skills to protect, not harm)
I also learned some basic Linux commands and how to use virtual machines. This class made me realize I want to go into cybersecurity as a career. It felt like solving mysteries and stopping bad guys online.
